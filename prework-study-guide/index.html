HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • You can use a as an anchor and link to other pages.
  • Elements that are nested within other elements are called child elements.
  • h1 through h6 represent the level of heading a given block of text represents
  • h1 should only appear on a page one time and its important to go in order from h1 to h6

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • When a developer wants to style an element directly in the HTML file, they use inline CSS
  • An example of inline CSS is using the style attribute
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main will pull down the latest main branch
  • when creating a new branch you should name it the feature you are working on

JavaScript

  • A variable is simply a named container that allows us to store data in our code
  • The .js file extension indicates a javascript file
  • We may see var from time to time as the keyword to create a variable
  • Const keyword allows us to create variables that can't be reassigned a value
  • We use an assignment variable like = to assign the specific data that we want stored in the variable
  • JavaScript reads from top to bottom as part of its control flow.
  • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • An array is a single variable that is used to hold a group of data
  • A single array can store different types of data, such as strings, numbers, and so on
  • In JavaScript and many other programming languages, arrays are zero-indexed and sequential
  • We use let, followed by a unique variable name and the assignment operator = to assign the specific data that we want stored in our variable
  • Unlike with const, let allows us to reassign the value of the variable over time.
  • A function is simply a set of instructions that tells the computer how to perform a certain task
